What Happens When You Open a Website?

  • You type a website URL in the browser.
  • The browser sends a request to the server through the Internet.
  • The server processes the request and checks the database.
  • The server sends back the required files (HTML, CSS, images).
  • The browser displays the website on your screen.

    • Explanation of Client-Server Architecture

    Client-Server architecture is a model where the client (user device) sends a request to a server. The server processes the request and returns the response through the Internet.
